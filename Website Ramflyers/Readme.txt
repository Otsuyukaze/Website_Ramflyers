Agar Website bekerja secara maksimal, silahkan lakukan beberapa hal dibawah:

1. Download font tersebut
   https://fonts.google.com/specimen/Nunito+Sans

2. Set scale laptop/PC menjadi 125%

3. Set scale browser menjadi 100%

4. Location untuk file(Untuk Admin)
   File php
   ../ramflywebsite_Data/PHP/

   File mp4/m4a/wav
   ../ramflywebsite_Data/Assets/Audio/

   File png/jpg/jpeg (untuk membuat animasi)
   ../Assets/Image/Animated/

   File png/jpg/jpeg (untuk background)
   ../Assets/Image/Backgrounds/

   File png/jpg/jpeg (untuk tombol)
   ../Assets/Image/Buttons/

   File png/jpg/jpeg (untuk tombol logo)
   ../Assets/Image/Buttons 2.0/
